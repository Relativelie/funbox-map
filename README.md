# Map with yandex.maps
## Building
1. Clone repository
``git clone git@github.com:Relativelie/funbox-map.git /path/to/map``
``cd  /path/to/map ``
2. Install dependencies
``npm install ``
4. Create file .env.defaults with yandex api key for example:
`` REACT_APP_API_KEY = '12345qwerty'``
3. Run project 
``npm start ``

## Testing
Running tests with: 
`` npm test``
